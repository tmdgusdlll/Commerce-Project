[프로젝트 기능]
- 각 카테고리별 상품이 나눠져있다.
- 장바구니에 상품을 추가할 수 있다 
재고 >= 입력수량 시 정상적으로 추가
재고 < 입력 수량 시 "재고 부족" 출력
- 장바구니에 상품이 담겨있다면 메인 화면 + 주문관리 화면을 출력한다
- 장바구이네 담긴 상품이 여러개일 때 각 상품별 정보를 띄우고, 총 (예상)주문금액도 출력해준다.
- 주문 확정시 주문한 수량 만큼 재고가 업데이트(차감)되며 장바구니도 초기화된다.
